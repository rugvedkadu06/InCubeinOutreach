# PostgreSQL + Docker Migration Plan

## Goal

Migrate the existing FastAPI + SQLite application to a production-ready architecture using:

- Docker
- Docker Compose
- PostgreSQL
- SQLAlchemy
- Alembic

## Objectives

- Replace SQLite with PostgreSQL
- Containerize backend and frontend
- Configure the application using environment variables
- Migrate existing data safely
- Keep the application functional throughout the migration

## Planned Milestones

1. Repository preparation
2. Docker setup
3. PostgreSQL integration
4. SQLAlchemy migration
5. Data migration
6. Backend refactoring
7. Frontend improvements
8. Production readiness
